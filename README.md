# מנוע מילוי טופס הפניקס — GitHub Pages

פרויקט סטטי מוכן ל־GitHub Pages.

## מה הוא עושה
- טוען טופס PDF מקורי של הפניקס שנמצא בתיקיית `templates/`.
- מקבל JSON ממערכת אחרת או מהדוח התפעולי.
- ממלא אוטומטית את מה שקיים.
- מוסיף שדות Editable על גבי ה־PDF לדברים שחסרים כרגע.
- מוריד למחשב PDF חדש.

## מבנה
- `index.html` — מסך העבודה
- `styles.css` — עיצוב
- `app.js` — מנוע המילוי בדפדפן
- `templates/GetPDF.pdf` — הטופס המקורי
- `examples/operational-report.sample.json` — דוגמת נתונים

## העלאה ל־GitHub Pages
1. פתח ריפו חדש ב־GitHub.
2. העלה את כל הקבצים כמו שהם.
3. ב־Settings > Pages בחר Deploy from a branch.
4. בחר branch ראשי ו־`/root`.
5. שמור.
6. פתח את הכתובת ש־GitHub ייתן לך.

## הערות
- זהו בסיס עבודה צד־לקוח בלבד, בלי Node ובלי שרת.
- המיפוי כרגע ראשוני ומוכן להמשך דיוק עמוד־עמוד.
- אם הטופס המקורי יוחלף, מספיק להחליף את הקובץ ב־`templates/GetPDF.pdf` ולעדכן קואורדינטות ב־`app.js`.
