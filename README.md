# INVESTO Assistant Repo

ריפו התחלתי עבור `INVESTO העוזר החכם`.

## מה יש בריפו
- `index.html` — מסך הצ'אט
- `styles.css` — עיצוב פרימיום של החלון
- `discount-rules.js` — דוגמאות חוקים להנחות
- `app.js` — לוגיקת צ'אט בסיסית

## ייעוד
הריפו נבנה כדי לעלות בנפרד מהריפו הראשי, ולהיטען בתוך חלון צף במערכת הראשית דרך iframe.

## תמיכה בפרמטרים מהמערכת הראשית
אפשר להעביר ב-URL:
- `agentName`
- `agentRole`

דוגמה:
`https://your-domain.example/investo/?agentName=אוריה&agentRole=ops`

## חשוב
1. מנוע רפואי כרגע הוא שלד בטוח בלבד, עם מידע כללי ודוגמאות.
2. מנוע ההנחות כרגע מבוסס על דוגמאות חוקים בלבד.
3. לפני פרודקשן צריך לחבר:
   - מקור ידע רפואי מאומת
   - טבלת הנחות אמתית לפי חברה / מוצר / גיל
   - לוגים / הרשאות / מעקב

## שלב הבא בריפו הראשי
להחליף את לחצן `צור ליד` ל-`INVESTO העוזר החכם`
ולפתוח iframe צף אל הכתובת של הריפו הזה.
